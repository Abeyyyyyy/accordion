# Accordion Component

This project is a simple Accordion built using HTML, CSS, and JavaScript.

## 🚀 Features

* Toggle accordion (open & close)
* Only one item open at a time
* Smooth animation
* Clean UI based on roadmap.sh mockup

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript (DOM Manipulation)

## 📂 How to Run

1. Clone this repository
2. Open `index.html` in your browser
